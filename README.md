Multi Disease Prediction System
Project Description

This is a Machine Learning based Python project that predicts diseases such as Heart Disease and Diabetes using patient medical data.

The program asks the user to enter patient details and then predicts whether the patient may have the disease.

This project was developed as part of an internship task.

Features

Predict Heart Disease

Predict Diabetes

Interactive command line interface

Allows checking multiple patients

Allows checking multiple diseases

Technologies Used

Python
Pandas
Scikit-learn
Machine Learning

Project Files

disease_prediction.py → Main program
heart.csv → Heart disease dataset
diabetes.csv → Diabetes dataset

Installation

First install required libraries using this command:

pip install pandas
pip install scikit-learn

Or install all libraries using:

pip install -r requirements.txt

How to Run

Open command prompt and go to the project folder:

cd Desktop
cd CodeAlpha_DiseasePrediction

Run the program:

python disease_prediction.py
